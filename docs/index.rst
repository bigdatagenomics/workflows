bdgenomics.workflows Documentation
==================

bdgenomics.workflows is an open-source collection of workflows for running
`Big Data Genomics`_ tools.

Check out our `website`_ for a comprehensive list of `Big Data Genomics`_ tools.
Feel free to also join us on `GitHub`_ and `Gitter`_. bdgenomics.workflows is
built on top of the `Toil`_ workflow management system and is designed to run
both locally and on the `Amazon Web Services`_ cloud.

.. _website: http://bdgenomics.org/
.. _Big Data Genomics: https://github.com/bigdatagenomics
.. _spot market: https://aws.amazon.com/ec2/spot/
.. _Amazon Web Services: https://aws.amazon.com/
.. _GitHub: https://github.com/bigdatagenomics/workflows
